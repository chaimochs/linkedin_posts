Two things I like to do when first installing Visual Studio Code:  
1. I find the Welcome screen that is shown every time the program begins annoying. In order to suppress it,  Uncheck “Show welcome page on startup” at the bottom of the Welcome page SEE SCREENSHOT. The page will not show again.  
Additionally, if you don’t want VSC to show release notes after an update, uncheck the text box on that page too SEE SCREENSHOT.  
2. Since we will be using Claude Code as our AI, we want to avoid having the default chat panel open at startup, because we can easily confuse it with Claude’s panel. To disable VSC’s default AI/chat in VS Code:  
- Open Settings (Code → Settings → Settings, or Ctrl+, [Windows] or CMD+, [MAC]).​   
- In the search box, type: @id:chat.disableAIFeatures.  
- Check Disable AI features (or similar checkbox that appears). SEE SCREENSHOT
- Restart VS Code so all chat/AI UI disappears.  
Now, we have a “minimalist” copy of VSC ready for our AI coding.
