CLAUDE.md—the “brains” behind your Claude Code projects.  
   
 CLAUDE.md is a simple markdows (formatted text) file which contains crucial setup information  for Claude Code.  
   
 There should be one CLAUDE.md file in the main folder of your computer’s installation (~/,YourUserName>/.claude in Mac; C:<YourUserName>.claude in Windows) and one in the root directory of each of your projects.  
   
 It serves as a persistent memory bank for Claude Code, allowing the AI to understand your project’s architecture, coding standards, and workflows without having to repeat instructions at every session.  
   
 Best practices:  
- Keep CLAUDE.md short and focused, with only the instructions Claude truly needs all the time. Instead of stuffing in every detail, point Claude to where it can find more information when needed.  
- Do not rely on it for jobs better handled by linters, formatters, hooks, or commands, and do not create it carelessly.
