Want to be on the cutting edge of AI automation?

Use Claude Code.

I will be using Claude Code in VS Code, and you should too. It allows you to review and edit the files that Claude is working on.

Go to the extensions tab on the VSC sidebar and look for the Claude Code extension. SEE DIAGRAM

Install the Claude Code extension. SEE DIAGRAM

Once Claude Code is installed you can click it by clicking on one of the icons on the left sidebar or on the top right of the VSC screen. SEE DIAGRAM

The first time you open Claude Code it will ask you to log in to your Claude Subscription. You need a paid Anthropic subscription to use Claude Code. If you will be doing AI automations, as I am, you will need a paid subscription (~$20/mo) in any case, otherwise you will be limited to several minutes of use a day. Login to or sign up for a Claude subscription.

Note: It is best to also install Claude in your native terminal. Open a terminal window (on Mac/Linux or PowerShell on Windows) and type:

Mac: curl -fsSL https://claude.ai/install.sh | bash

Linux: curl -fsSL https://claude.ai/install.sh | bash

Windows: irm https://claude.ai/install.ps1 | iex
